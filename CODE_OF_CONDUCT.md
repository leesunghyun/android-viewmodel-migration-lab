# Code of Conduct

This project is intended to be a friendly learning space.

## Expected behavior

- Be respectful, patient, and constructive in issues and reviews.
- Focus on code, behavior, and evidence.
- Be honest about uncertainty.
- Keep discussions inclusive and beginner-friendly.

## Unacceptable behavior

- Personal attacks, harassment, discrimination, or hate language.
- Repeatedly disruptive behavior.
- Sharing private data, credentials, or sensitive repository information.

## Moderation process

1. Warnings are shared by maintainer comments.
2. Repeated issues may lead to temporary restrictions.
3. Serious violations can be blocked from issues/PR creation.
4. Appeals are accepted via a new issue with a short remediation plan.

If you are unsure, open an issue with context and expected behavior.
